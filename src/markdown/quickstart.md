### 介绍

通过本章节你可以了解到 @vcomponetns/vant-ui 的安装方法和基本使用姿势。

### 前置条件

```js
// main.js
// 建议全局引入
import Vue from 'vue'

import Vant from 'vant'
import 'vant/lib/index.css'

Vue.use(Vant)
```

### 安装

```shell
npm i vantui-components -S
```

### 引入组件

```js
// main.js
// 全局引入
import VComponents from 'vantui-components'

Vue.use(VComponents)
```
